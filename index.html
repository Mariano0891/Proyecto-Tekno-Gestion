¿Que es Tekno?

Tekno gestión es un software ideal para la administración integral de empresas de diversos rubros. Sus múltiples módulos son adaptables y personalizables para cualquier actividad convirtiéndolo así en un sistema muy versátil.

Entre sus funciones/módulos podemos encontrar Ventas, Compras, Stocks, Caja/Bancos, Contabilidad, Sueldos, Servicios, etc. Los mismos pueden adquirirse con el sistema completo o individualmente, dependiendo de las necesidades y/o requerimientos de su empresa.

El sistema es compatible con todas las versiones de Windows, sean de 32 o 64 bits, y dispone de integración con diversas herramientas como Word, Excel, Outlook. También ofrece la posibilidad de trabajar con una gran variedad de archivos sea TXT, XLS, CSV, PDF, etc.
